# 스프링 프로젝트 8단계

+ TDD 코드 작성

+ REST API 적용

+ TDD 기반 리팩토링 적용

+ Repository 도메인 추가를 통한 모듈화

+ 인터페이스를 이용한 객체 의존성 주입

+ 레이어 분리
  
+ **Mock 오브젝트(가짜 객체)를 통한 테스트 코드의 의존성 낮추기**
  
  **Mockito 프레임워크를 이용**
  
  
  
  Spring은 POJO 사용을 권장
